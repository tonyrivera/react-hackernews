# react-hackernews
Hackernews implemented in React

# In terminal
# Step 1: clone repo
`git clone https://github.com/tonyrivera/react-hackernews.git`

# Step 2:
`cd react-hackernews/`

# Step 3:
`npm install`

# Step 4:
`npm run build`

# Step 5:
Navigate browser to provided url:port
